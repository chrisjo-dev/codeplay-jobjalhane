## 0. 목적

기존 진로·적성 검사의 해석 편차, 실행 전략 부재, 대규모 제공의 어려움을 해결하기 위해 AI 기반으로 **일관된 전문가 수준 분석**과 **실행 가능한 커리어 로드맵**을 제공하는 서비스의 기획 방향을 정리한다.

---

## 1.서비스 개요

이 서비스는 사용자의 심리·흥미·성향·적성 데이터를 기반으로 **직업적 정체성 프로파일**, **핵심 강점·약점**, **직무 적합도**, 그리고 **향후 3년간의 커리어 로드맵**을 자동으로 생성해주는 AI 기반 진로 설계 솔루션이다. 전통적인 검사 해석을 넘어 실제 직무 선택과 이직 전략에 바로 적용 가능한 실행 계획을 제공한다.
AI는 입력된 검사 데이터(RIASEC, Big5/MBTI, 적성, 생활사), 학력·경력 정보, 기술 스택 등을 기반으로 개인의 **직업적 정체성**, **강점·약점**, **적합 직무군**, **부적합 환경**, 그리고 **향후 3년 커리어 전략**을 구조화된 보고서 형태로 제공한다.

---

## 2. 문제 정의

현실에서 사용자가 겪는 문제는 다음 네 가지로 요약된다.

### 2.1 검사 결과를 읽어도 방향을 못 잡는다

- RIASEC, Big5, 적성검사 등 각종 결과가 제공되지만 사용자는 결과를 **직무 선택 기준**으로 연결하지 못한다.

### **2.2 전문가는 많지만 “고품질 분석을 저렴하게 대량 제공”하는 것은 불가능하다**

- 대부분의 진로상담은 검사 해석 → 설명
- “이제 무엇을 해야 하는가”에 대한 전략은 제공되지 않는다.

### 2.3 상담 품질이 편차 심하다

- 상담사·기관마다 해석 방식·조언 수준·전략 정확도가 크게 다르다. 일관된 서비스가 어렵다.

### 2.4 사용자는 진로 선택에 구조적 기준이 없다

- 대학생, 취준생, 커리어 전환자는 본인의 강점·약점·성향·리스크를 직업군과 연결하는 프레임이 없다. 결국 감으로 선택하거나 주변 조언에 의존한다.

---

## 3. 왜 AI가 필요한가

- **일관성 확보**: 동일 기준의 분석 모델을 통해 상담사 편차 없이 안정적 품질 제공.
- **전문성 강화**: 심리·성향·적성·직무 데이터를 통합한 복합 분석을 자동화해 전문가 수준의 판단을 제공.
- **대량 처리 가능**: 대학·기관 등 대규모 사용자 집단에 즉시 적용 가능.
- **전략 자동화**: 직무 요구사항, 산업 패턴, 성장 단계까지 반영한 로드맵 생성은 AI가 가장 효율적.

---

## 4. 주 타켓층

- **대학생·취준생(20대) -** 직무 선택 기준 부재, 검사 해석 어려움.
- **사회초년생·경단기 구직자 -** 강점 정리·전략 설계 수요.
- **대학·교육기관(B2B) -** 대량 검사 해석 자동화 및 진로 상담 효율화.

## 5. 서비스 유저 FLOW

검사결과  PDF 업로드 → 2) 데이터 정규화 → 3) AI 분석 → 4) 정체성·강점·약점 도출 → 5) 직무군 및 부적합 환경 제공 → 6) 3년 로드맵 생성 → 7) 자소서 문장 제시 → 8) PDF 보고서 출력

---

## 6. 기술 구조

- LLM 기반 분석 및 문장 생성
- 성향·적성 데이터 벡터화
- 직업군 유사도 계산 모델
- 경력 단계별 로드맵 생성 알고리즘
- PDF 자동 리포트 생성 모듈

## 7. 기대 효과

### 개인

- 직무 선택 정확도 향상
- 이직·전환 실패 가능성 감소
- 자소서·면접 준비 효율 증가

### 기관

- 상담 품질 표준화
- 상담 시간 절감
- 대규모 진로 데이터 기반 의사결정 가능
